# 📦 Supply Chain & Inventory Health Dashboard (PoC)

## Overview
This repository contains a Proof of Concept (PoC) web application designed for supply chain consultants and warehouse managers. It provides real-time visibility into inventory health, highlights critical stockouts, and tracks key logistics KPIs to optimize warehouse operations.

## Key Features
* **Live KPI Tracking:** Monitor Total SKUs, Units in Facility, and SKUs at Risk.
* **Interactive Visualizations:** Dynamic charts comparing Stock on Hand vs. Reorder Points.
* **Cloud Database Integration:** Fetches live data seamlessly from Google Sheets via Pandas.
* **Zero-Setup Access:** Hosted on Streamlit Community Cloud for instant mobile and web access.

## Tech Stack
* **Language:** Python
* **Frontend Framework:** Streamlit
* **Data Processing:** Pandas
* **Database:** Google Sheets

## Business Value
This tool replaces static, manual spreadsheets with an automated pipeline, allowing operations teams to shift their focus from data entry to proactive decision-making.
